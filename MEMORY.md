# 《羽球一家人》项目进展总结 (2026-01-30)

## 项目状态
- **核心架构**：Node.js (Express) 后端 + Vue 3 (Vite) 前端 + SQLite 数据库。
- **部署情况**：已通过 PM2 托管，公网访问地址 `http://43.165.0.210:3000`。
- **GitHub 仓库**：`chzisnull/badminton-family`，已同步最新代码。

## 已实现功能
1. **赛制算法**：支持单打循环、固搭循环（两人一组配对）、多人轮转（平衡算法）。
2. **人员管理**：持久化存储选手档案，支持点击选择历史人员参赛，自动统计个人胜率。
3. **比赛详情**：实时计分、进度条显示、多选筛选人员对局、自动重定向已删除比赛至首页。
4. **管理功能**：支持删除比赛、重设赛制参数（轮次、类型）。

## 待办事项
1. **小程序重构**：计划将现有 Vue 3 代码迁移至 `uni-app` 以支持微信小程序。
2. **域名/HTTPS**：小程序上线需要备案域名和正式 SSL 证书。
